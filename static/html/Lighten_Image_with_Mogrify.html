Hey Jack, to lighten an image using the `mogrify` command, you can use the `-modulate` option to adjust the brightness. Here's a complete example with logging for you:

```bash
#!/bin/bash
# This script lightens an image using mogrify

# Define the input and output file paths
input_file="input.jpg"
output_file="output.jpg"

# Define the brightness level you want (100% means no change, >100% to lighten)
brightness_level=150%

# Log the operation
echo "Lightening image: $input_file -> $output_file with brightness: $brightness_level"

# Use mogrify to adjust the brightness
mogrify -modulate $brightness_level "$input_file"

# Log completion
echo "Image lightening complete. Check $output_file."
```

Make sure to replace `"input.jpg"` and `"output.jpg"` with your actual input and output file paths. You can adjust the `brightness_level` variable to control how much you want to lighten the image (e.g., `200%` for a stronger effect).

Save this script to a `.sh` file and execute it in your terminal. It will take the input image, lighten it, and save the result as the output image. Don't forget to make the script executable using `chmod +x script.sh` if needed.
